Requiremenet :
- java 7+ installed

Run application either by double click on the jar file or in a terminal
$ java -jar openworld-desktop-0.0.4-SNAPSHOT.jar

Controls :
- Z : move forward
- Q : move left
- D : move right
- S : move backward
- SPACE : toggle 2D map
- RIGHT BUTTON + MOUSE MOVE : rotate the camera
