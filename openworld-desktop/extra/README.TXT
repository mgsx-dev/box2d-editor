Requirements :
- java 7+
- Puredata vanilla 0.47.1

Run application either by double click on the jar file or in a terminal
$ java -jar openworld-desktop-0.0.4-SNAPSHOT.jar

Controls :
- Z : move forward
- Q : move left
- D : move right
- S : move backward
- SPACE : toggle 2D map
- Mouse Drag with LEFT Button : rotates the camera

Run Puredata and open both network.pd and openworld.pd patches.
You should see things happening in the openworld.pd patch...
